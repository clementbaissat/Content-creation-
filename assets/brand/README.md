# HopeStage Brand Assets

Canonical local brand assets for HopeStage image generation, thumbnails, templates, and design work.

These files are stored here so future content workflows can reference the exact approved logo and icon files from the repository instead of external desktop paths.

## Files

- `Frame 10.png`
- `hopestage 256.png`
- `HopeStage google logo.png`
- `HopeStage logo home.png`

## Usage

- Use these files as the default reference assets when creating thumbnails, social visuals, mockups, or brand-consistent image prompts.
- Do not redraw or reinterpret these assets when the request is to use the official logo or icon.
- Prefer referencing the exact file path from this folder in future workflows.

## Notes

- `hopestage 256.png` is the icon mark.
- `HopeStage google logo.png` is the square logo lockup on dark background.
- `HopeStage logo home.png` is the horizontal wordmark lockup.
- `Frame 10.png` is the pale-background brand lockup.
